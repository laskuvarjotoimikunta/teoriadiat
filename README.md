# KIRJOITTAMINEN

## Tekninen toteutus

Nämä diat kirjoitetaan Markdown -muodossa. Koska lopullinen esitys tapahtuu
jonkin tulkin lävitse on tärkeää, että kirjoitusasu noudattaa suhteellisen
tiukkaa formaattia.

Aloita uusi kokonaisuus otsikkotasolla yksi, kirjoittamalla rivin alkuun '#'. Tämän jälkeen tulevat diojen otsikot ovat otsikkotasoa kaksi; rivin alkuun '##'.

### Korostukset

Kun haluat erityisesti korostaa jotakin laita rivin alkuun

```
.red[
```

Lopeta korostus laittamalla uuden rivin alkuun ']'. Käytä tätä harkiten, sillä koko Markdownin idea on erottaa sisältö ja muotoilu.

### Linkit videoihin

Voit lisätä linkin Youtube-videoon helposti laittamalla omalle rivilleen sulkujen sisään vidon ID-koodin seuraavasti:

```
![:ytvideo](VIDEO_ID)
```
